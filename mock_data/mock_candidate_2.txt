Q2: What is the difference between frontend and backend?
A:  Frontend deals with the presentation layer — HTML, CSS, JavaScript, and
    frameworks like React or Angular. It runs in the user's browser and is
    responsible for what the user sees and how they interact with the product.
    Backend runs on a server and handles things like authentication, data
    processing, business rules, and database access. It exposes APIs that the
    frontend consumes. Both are equally important and need to be designed well
    for an application to perform reliably.
